<h1>JS Pong (JavaScript Tutorial)</h1>

### There is a "JavaScript Tutorial" that I followed to code Pong


---

### Steps

0. Clone the repo from GitHub : `git clone git@github.com:jaldabaoth-code/JS-Pong.git`
1. Enter the directory : `cd JS-Pong`
2. Run the internal PHP webserver with `php -S localhost:8000 -t public`
3. Go to <b>localhost:8000</b> with your favorite browser
